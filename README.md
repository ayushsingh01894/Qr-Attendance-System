# QR Code Attendance System

A real-time attendance system where teachers generate a QR code and students mark attendance using their mobile devices.

## Features
- QR based attendance
- Mobile support using ngrok
- CSV attendance export
- Duplicate attendance prevention

## Tech Stack
- Python
- Flask
- MongoDB
- HTML/CSS

## 🎯 Project Motivation

Manual attendance systems are time-consuming and error-prone.  
This project automates attendance using QR codes, reducing manual effort and improving accuracy in academic environments.

## 🔁 Application Workflow

1. Admin generates a QR code for a session
2. Students scan the QR code
3. Attendance data is stored in the database
4. Admin views attendance reports in the dashboard

## ⚠️ Current Limitations

- QR code expires only on page refresh
- No facial verification (future scope)
- Designed for small to medium classrooms
